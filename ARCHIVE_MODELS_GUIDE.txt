Archived Model Files Guide

The submitted project already includes the active model checkpoints required to run the system under:

models/

The archived checkpoints were uploaded separately to Google Drive because they are large and mostly represent earlier experiments, alternative architectures, last-epoch checkpoints, and SAR model variants. These archived files are not required for the quick test in setup.txt, but they are provided for reproducibility and detailed review.

Google Drive archive link:
https://drive.google.com/drive/folders/1aS0zhJi3q_7pA_lezm6jyXD5BsmMv9Tx?usp=sharing


1. Active checkpoints included in this submission

These are the default checkpoints used by the current project pipeline:

- models/classifiers/noise_classifier_best.pt
  Main noise classifier. Predicts the input noise type: gaussian, salt_pepper, speckle, or periodic.

- models/denoisers/gaussian/gaussian_unet_best.pt
  Active Gaussian noise denoiser.

- models/denoisers/salt_pepper/salt_pepper_unet_best.pt
  Active salt-and-pepper noise denoiser.

- models/denoisers/speckle/speckle_unet_residual_hybrid_best.pt
  Active speckle noise denoiser.

- models/denoisers/periodic/periodic_fft_guided_nafnet_best.pt
  Active periodic noise denoiser.

- models/denoisers/periodic/periodic_dual_branch_unet_best.pt
  Additional periodic denoiser checkpoint kept for comparison and fallback testing.

- models/denoisers/sar/best_noise_map_sar_model.pth
  Active SAR noise-map model used for the SAR output samples.

4. How to use an archived checkpoint

Archived checkpoints should only be used if the corresponding script and architecture match the checkpoint type.

Examples:

- RGB denoiser checkpoints can be tested with scripts under:
  scripts/evaluation/
  scripts/training/

- SAR noise-map checkpoints can be tested with:
  scripts/sar/inference/run_noise_map_inference.py
  scripts/sar/inference/run_noise_map_residual_cnn_inference.py

If an archived checkpoint is used, place it in the expected path or update the --model-path argument in the related script.


5. Important note for reviewers
